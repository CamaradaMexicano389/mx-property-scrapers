# mx-property-scrapers

Conectores de extracción de datos inmobiliarios públicos de México.

Repositorio comunitario independiente. Estos conectores no forman parte del
estándar PADIM, del paquete pip ni de ninguna organización institucional.

## Stack

- Python 3.11+
- curl_cffi (Vivanuncios)
- Scrapling (Inmuebles24)
- PostgreSQL (almacenamiento)
